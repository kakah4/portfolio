# 💰 Expense Tracker

A clean and responsive expense tracking app built with Next.js and TypeScript. Track your income and expenses, visualize spending by category, and stay on top of your finances.

## 🔗 Live Demo
[expense-tracker-trkn.vercel.app](https://expense-tracker-trkn.vercel.app/)

## ✨ Features
- Add income and expense transactions
- View real-time balance, total income, and total expenses
- Pie chart breakdown of expenses by category
- Delete transactions instantly
- Fully responsive design

## 🛠️ Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts

## 🚀 Getting Started
```bash
git clone https://github.com/kakah4/expense-tracker.git
cd expense-tracker
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
